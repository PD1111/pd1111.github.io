  Japanese Mahjong Score Calculator  (Personal Project)

Japanese Mahjong is usually played with 136 tiles with a series of complicated rules of determining the scores of each round. MahjongCal is an android app for calculating the score automatically base on the winning combination of 14 tiles. Peter is currently working on improving the UI design as well as the functionality of the app

  BlueTeam CustomUI  (Work Project)

BlueTeam CustomUI is designed for customers to create their own web pages. First, user use the desktop application to create a tree diagram which represents the web page they are going to create, then custom information will be stored into databse. Finally, the web application will then render the whole page on page-load based on data stored in database.
